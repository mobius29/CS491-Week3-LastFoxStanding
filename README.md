## 게임 목표
착한 여우들 사이에 숨어있는 나쁜 여우(다른 플레이어)를 찾아 포도를 던지세요

## 조작 방식
이동 : wasd
포도 던지기 : ijkl

## 팁

꽃게가 나와 춤을 출 때 착한 여우들은 모두 같은 춤을 따라 춥니다. 다른 행동을 하고 있는 나쁜 여우를 찾으세요
